NAME

r.pixelarea - Creates a raster showing the area of each pixel.

KEYWORDS

raster

SYNOPSIS

r.pixelarea
r.pixelarea help
r.pixelarea [zfactor=float] output=name [--overwrite] [--verbose] [--quiet]

Flags:

--overwrite
Allow output files to overwrite existing files
--verbose
Verbose module output
--quiet
Quiet module output

Parameters:

zfactor=float
Multiplicative factor to convert from meters to desired units: default = 0.0001 for ha
Default: 0.0001
output=name
Name for output raster map
