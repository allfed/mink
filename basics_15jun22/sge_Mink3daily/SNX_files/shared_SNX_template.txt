! Comments are indicated with the "!"
! 
! This is a custom template intended to be read in by a python script (generate_SNX_files.py)
! The template is used to generate all the SNX files. SNX files are read in by DSSAT to control
! the simulation. other .**X files can be used by DSSAT as well, but we use SNX for gridded approach.
! 
! Data specific to each SNX file is stored in data_to_generate_SNX_files.csv
! The generated SNX files are placed in the generated_SNX folder.
! NEW BLOCK contains a new block of text to fill in the SNX file.
! Then right after NEW BLOCK the variable name is given.
! This variable name will be the header in the data_to_generate_SNX_files.csv
! 
! The {[variable]: >[number]} pattern is making sure the generated SNX properly fills in
! a [number] wide series of characters in the SNX, with space padded in front. Python does it.
! 
! Some placeholders are filled in on a by-pixel basis in java (Mink3p2daily.java).
! These placeholders are:soil ID (ssssssssss) , start planting date (ppppS),
! end allowed planting date (ppppE), random seed (rrrrr), number of years simulated (nnnnn),
! environmental modification date (iiiiS), and weather station code (wwww).
! Also, ___place initializations here___, and ___place fertilizers here___ are filled in by java.
